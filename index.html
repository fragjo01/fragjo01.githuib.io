Projects

Professional Tennis Match Analysis

This project analyzes professional tennis match data to predict match duration using statistical models. We considered aces, double faults, service/return points, total points, and court surface.

Final model: Stepwise second-order interaction model with lowest Test MSE (~0.897).

Sports Car Price Analysis

This project analyzes the performance vs. price trade-off in high-end sports cars using R and Jupyter notebooks. We explore how efficiently each car delivers horsepower, torque, and acceleration relative to its price.

Outputs include cleaned dataset and visual charts. Key insights: some cars deliver higher performance per dollar, and acceleration, horsepower, and torque are not always aligned with price.

Tools: R (tidyverse, ggplot2), Jupyter with R kernel

Stock Forecast Project

This project forecasts stock prices for AAPL, MSFT, AMZN, and GOOGL using SQL, Python (Prophet), and Power BI. We used SQL to retrieve data, Prophet to forecast 1-year prices, and created visual comparisons and bar charts to track expected growth. Optional dashboards were created in Power BI.

Tools: SQL, Python (Prophet), Power BI
