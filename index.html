Tennis Match Data Analysis

This comprehensive tennis analytics project demonstrates my ability to handle real-world sports data from collection through deployment. I developed an ETL pipeline to clean and structure match-level and point-by-point data, then engineered performance metrics including serve efficiency, return ratings, rally patterns, and winner-to-error ratios. Using exploratory data analysis with Matplotlib and Seaborn, I identified key performance indicators that separate top players from the field. I built a machine learning model using scikit-learn to predict match outcomes with 78% accuracy, leveraging features like momentum shifts and surface-specific performance. The final deliverable includes an interactive Flask dashboard for real-time player comparisons, fully documented Jupyter notebooks for reproducibility, and a comprehensive technical report with actionable insights for coaches and analysts.

  • Role: Data Analyst & ML Engineer
  • Duration: Mar 2025 — Nov 2025
  • Tech: Python, Pandas, Scikit-learn, Flask, SQL, Docker
