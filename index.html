Tennis Analytics — Case Study

How I Did It:

This project analyzes over 350,000 professional tennis matches from ATP and ITF circuits to identify key factors influencing match duration. Using match-level statistics including total points played, number of sets, aces, double faults, and court surface, I developed a linear regression model. To address skewness in match duration, I applied a log transformation and assessed model performance through residual diagnostics and cross-validation. Stepwise selection and interaction effects were incorporated to refine the model. I evaluated the model on six representative matches showing predicted durations and prediction intervals, and confirmed that across the dataset, 99.63% of observed durations fell within their prediction intervals. Findings highlighted that total points played is the strongest predictor, aces and double faults tend to shorten matches, and clay courts generally produce longer matches than hard or grass surfaces.

Sports Car Price Analysis

How I Did It:

This project analyzed the performance vs. price trade-off in high-end sports cars using R. I started by cleaning and processing the raw dataset, then engineered features to evaluate horsepower, torque, and acceleration relative to price. Visualizations were created using ggplot2 in Jupyter notebooks with R kernel to identify cars delivering higher performance per dollar. I generated charts such as Speed per Dollar, Horsepower per Dollar, and Torque per Dollar, ranking the top 20 best-value sports cars. The final outputs included processed datasets and visual charts stored in CSV and PDF formats, providing actionable insights for evaluating car performance efficiency relative to cost.

Stock Forecast Project

How I Did It:

For this project, I used SQL to filter stock data for AAPL, MSFT, AMZN, and GOOGL, then applied Python with Prophet to forecast prices for 1 year. I visualized forecasts vs. actual stock prices, created bar charts to compare expected growth, and optionally developed dashboards in Power BI for interactive insights. All analysis and visualizations were organized in Jupyter notebooks, and outputs including charts and forecast files were stored in structured folders for easy access and review.
