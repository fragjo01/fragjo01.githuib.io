Tennis Analytics — Case Study

This project explores match duration in professional tennis using over 345,000 matches from various tournaments. The goal was to understand how key in-match statistics influence match length and to develop predictive models for match duration. Project Highlights: Response Variable: Match Duration (minutes) Predictor Variables: Aces, Double Faults, Service Points Attempted, Return Points Attempted, Total Points, Court Surface Data Summary: Dataset includes 345,974 matches, filtered for matches between 35 and 300 minutes; point-based variables are strongly correlated, while aces and double faults are weaker predictors. Modeling Approach: First-Order Model: All predictors statistically significant; Adjusted R² = 0.835, Residual SE = 14.19 min. Interaction Model & Stepwise Regression: Added two-way interactions between quantitative variables; final model includes 13 terms selected via AIC stepwise regression, Adjusted R² = 0.8309, improved fit and clarity. Transformed Model: Box-Cox transformation applied to response variable for normality; all predictors remain significant, Adjusted R² = 0.829. Model Evaluation: Compared multiple models using Test Mean Squared Error (MSE); the stepwise interaction model achieved the lowest Test MSE (~0.897), chosen as the final model. Key Insights: Aces and double faults negatively impact match duration. Interactions between service points and aces, as well as return points and total points, significantly improve predictive power. Most matches are short and right-skewed; transformations improve model accuracy. Tools Used: R (tidyverse, ggplot2) Jupyter Notebook with R kernel Statistical modeling, stepwise regression, and data visualization.

Sports Car Price Analysis

This project examines the performance-to-price trade-offs in high-end sports cars using R. I analyzed how efficiently each car delivers horsepower, torque, and acceleration relative to its cost, identifying top value performers. Project Highlights: Data & Workflow: Cleaned dataset (processed_data.csv) and visualized insights using R notebooks. Visualizations: Generated charts for Speed ​​per Dollar, Horsepower per Dollar, and Torque per Dollar (PDF). Key Insights: Not all high-priced cars provide proportionally higher performance; top 20 rankings highlight the best value sports cars. Tools: R (tidyverse, ggplot2) and Jupyter Notebook with R kernel for reproducible analysis.
